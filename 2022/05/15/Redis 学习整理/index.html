redis学习整理

Redis 简介

  1. 一个高性能的 key-value 数据库。
  2. 原子 – Redis的所有操作都是原子性的，意思就是要么成功执行要么失败完全不执行。

Redis 键(key)

语法

Redis 键相关的基本命令：

  1. DEL key
    该命令用于在 key 存在时删除 key。

  2. EXISTS key
    检查给定 key 是否存在。

  3. EXPIRE key seconds

    为给定 key 设置过期时间，以秒计。

  4. TTL key
    以秒为单位，返回给定 key 的剩余生存时间(TTL, time to live)。

  5. RENAME key newkey
    修改 key 的名称

  6. TYPE key
    返回 key 所储存的值的类型。

Redis 字符串(String)

语法

Redis 字符串命令

  1. SET key value
    设置指定 key 的值。

  2. GET key
    获取指定 key 的值。

  3. MSET key value [key value …]
    同时设置一个或多个 key-value 对。

  4. MGET key1 [key2..]
    获取所有(一个或多个)给定 key 的值。

  5. STRLEN key

返回 key 所储存的字符串值的长度。

  1. APPEND key value

如果 key 已经存在并且是一个字符串， APPEND 命令将指定的 value 追加到该 key 原来值（value）的末尾。

Redis 哈希(Hash)

实例

redis hash 基本的相关命令：

  1. HSET key field value

将哈希表 key 中的字段 field 的值设为 value 。

  1. HDEL key field1 [field2]
    删除一个或多个哈希表字段

  2. HGET key field
    获取存储在哈希表中指定字段的值。

  3. HGETALL key

获取在哈希表中指定 key 的所有字段和值

  1. HDEL key field1 [field2]
    删除一个或多个哈希表字段

  2. HSCAN key cursor [MATCH pattern] [COUNT count]

迭代哈希表中的键值对。

Redis 列表(List)

实例

列表相关的基本命令：

  1. RPUSH key value1 [value2]
    在列表中添加一个或多个值

  2. LPUSH key value1 [value2]

    将一个或多个值插入到列表头部

  3. LINSERT key BEFORE|AFTER pivot value
    在列表的元素前或者后插入元素

  4. LINDEX key index
    通过索引获取列表中的元素

  5. LRANGE key start stop
    获取列表指定范围内的元素

  6. LREM key count value
    移除列表元素

Redis 集合(Set)

实例

Redis 集合基本命令：

  1. SADD key member1 [member2]

    向集合添加一个或多个成员

  2. SMEMBERS key

返回集合中的所有成员

  1. SINTER key1 [key2]

    返回给定所有集合的交集

  2. SDIFF key1 [key2]
    返回第一个集合与其他集合之间的差异。

  3. SCARD key

获取集合的成员数

  1. SSCAN key cursor [MATCH pattern] [COUNT count]
    迭代集合中的元素

Redis 有序集合(sorted set)

每个元素都会关联一个 double 类型的分数。redis 正是通过分数来为集合中的成员进行从小到大的排序。

有序集合的成员是唯一的,但分数(score)却可以重复。

实例

redis 有序集合的基本命令:

  1. ZADD key score1 member1 [score2 member2]
    向有序集合添加一个或多个成员，或者更新已存在成员的分数

  2. ZINCRBY key increment member
    有序集合中对指定成员的分数加上增量 increment

  3. ZCARD key
    获取有序集合的成员数

  4. ZRANGE key start stop [WITHSCORES]
    通过索引区间返回有序集合指定区间内的成员

  5. ZRANGEBYSCORE key min max [WITHSCORES] [LIMIT]

通过分数返回有序集合指定区间内的成员

  1. ZREM key member [member …]
    移除有序集合中的一个或多个成员
